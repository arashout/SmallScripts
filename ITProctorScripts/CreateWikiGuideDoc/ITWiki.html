File Paths

If the candidate has passed his/her Securexam(CPA) qualification exam, then chances are good that he/she will not experience difficulties during the exam. However, the Securexam(CPA) qualification exam is not a 100% indicator of a laptop's health, so problems can still occur.

File Handling

Securexam(CPA), when operating properly, will save the candidate's files to the USB key every minute.

If the candidate's computer suffers an unrecoverable hardware error, the files will be retrievable from the USB key.

If Securexam(CPA) is having issues with the USB key (i.e. cannot find it, or cannot write to it), Securexam(CPA) will deposit the encrypted file on the candidate's Desktop. The name of the file will look similar to this:

CPA_Oct26Exam_4012345_20131026_1537.ssi-response

If you cannot find the ssi-response file on the USB key, then it should be on the Desktop. If you cannot find it on the Desktop, check that the folder and search options view is set to show hidden files and folders. If you still cannot find them on the Desktop you will be able to find them archived.

Secondary Back-up Files

Securexam saves once a minute to the following folder:

C:\Users\(Your User Profile name)\AppData\Local\SSI\(your candidate ID)\(Exam name)

Some of the folders in the destination above may be hidden.  To view, you must display hidden files and folders by editing folder options.

There should be one subfolder for the Word portion and one subfolder for the Excel portion in the (Exam name) folder. Inside these folders are Securexam's automatic backup saves (saved once per minute). If you sort this folder by "Date Modified" you should see the most recently saved backup exam file for each subfolder, which should also correspond to the largest file size in the folder. If you see that the most recent file is *not* the largest file, it may be that the full exam was contained in one of the larger files saved a minute or two before the end of the exam.

These secondary backups can be decrypted using Workbench to yield the candidate's response files.

If a candidate experiences a problem during the exam process, there are a few points the IT Proctor should be aware of:

Time is of the essence!

During the exam, time is of the essence for the candidate. If there is a technical issue, the candidate should be instructed to start hand writing immediately while you try to solve the problem. Bring paper, a bubble sheet (for MCQ input), and a form for note-taking with you when you go to help a candidate. If a problem cannot be solved quickly (rule of thumb is usually within ten minutes), then the candidate will be required to write the remainder of their exam by hand.

IT Proctors are not hardware technicians

Keeping the "time is of the essence" idiom in mind, IT Proctors are not expected to perform intensive diagnostics of a candidate's laptop if it fails. This is a job for the candidate's IT department.

Keep calm

The candidate needs to feel assured that you know what you are doing. Any uncertainty projected by you can undermine the candidate's confidence.

Remember: The worst case scenario is that, after ten minutes, you will recover and print out the candidate's response and the candidate will write the balance of the exam by hand.  We have tracked candidates who wrote by hand and there is no correlation between writing by hand and failing.

You are not alone!

You are not the Presiding Officer

Apart from IT help, you are not to provide any other assistance to candidates. Candidates with any non-technical questions must be directed to an invigilator or the PO.

The PO is responsible for deciding about assigning extra time to candidates if they have technical difficulties. You may be asked for advice, but please understand that it is the PO who will make the final call.

Remote Desktop enabled

Securexam will refuse to start if it detects that Remote Desktop is enabled on the computer.

1) To disable Remote Desktop:

Windows XP:
•    Right-click the My Computer icon.
•    Select the Properties command.
•    Select the Remote tab.
•    Remove the checkmarks under Remote Assistance and Remote Desktop. (Some computers may not display the “Remote Desktop” check box.)
•    Click OK.

Windows Vista, Windows 7/8:
•    Right-click the Computer icon on the Start Menu.
•    Select the Properties command.
•    Click the Remote settings task.
•    Click Continue if necessary.
•    Remove the checkmark under Remote Assistance.
•    If there is a “Remote Desktop” section, select the Don’t allow connections to this computer option.
•    Click OK.

2) If Securexam continues to detect Remote Desktop, it will be necessary to disable the Remote Desktop services:

•    Click the Start Menu.
•    In Windows XP , select the Run command.
•    In Windows Vista , 7/8 , type the command into the Start search field.
•    Type the command “Services.msc” and press the Enter key.
•    Click Continue if necessary.
•    Right-click the Remote Access Connection Manager entry.
•    Select the Properties command.
•    Change Startup type to Disabled.
•    Click OK to close the Properties window.
•    Right-click the Terminal Services entry.
•    Select the Properties command.
•    Change Startup type to Disabled.
•    Click OK to close the Properties window.
•    Restart the computer.

At this point, if Securexam is still displaying messages about Remote Desktop, it is possible that the message is actually being caused by a problem with the laptop's power management system. The user should unplug the power cord to the laptop and try running Securexam again. Then plug the power cord back in.

What is Securexam?

Securexam is “lockdown” software that allows candidates to use their laptops in an environment that prohibits access to certain functions, files, and the Internet to ensure all candidates are working from a level playing field.

In order for candidates to use their laptops for any CPA evaluation, they must have previously installed Securexam(CPA), run a qualification exam, and had their laptop approved by Securexam. This process assists in finding and solving any laptop glitches prior to an exam.

Securexam does not support Macintosh computers – even if "Boot Camp" is installed. Candidates are not allowed to use Macs to write the exams.  Securexam also does not support tablets - this includes Surface machines if the keyboard is detachable.

Workbench Guide

The only time you'll need to use Workbench is when a candidate's laptop fails during the evaluation and s/he has a partial response that needs to be printed off so s/he can continue writing on either a backup laptop or by hand.

The Workbench printer will print to your default printer.

Setting the Print Options

This step is only required once. After the options are selected, they will be saved and will apply to all future print jobs.

Click the "Exam Printer" tab and then the "Print" button to open up the main application window.

To review the printer options, click the double arrow located on the left side of Securexam Workbench as illustrated below:


The exam printing pane on the left side will open and will allow you to choose all your required printing options.

Directory Print Options

For Directory Print Options, you can decide to pick either all files in a folder or individual files.

By default, the print option has been set to "SelectFromFolder".  This option allows you to print all files in a folder.

To print individual files, choose "SelectFromFileBrowser".

Once you have set your printing options, click the double arrow on the left side of the page to close the Exam Printing screen.

Selection Options

There are a number of printing options specified in the top bar of the print screen.

Selection Option Description
Search Subfolders If checked, the print utility will search the selected folder and all its sub-folders for exam files to display.
Otherwise, only files from the selected folder are displayed.
Auto Print Selected Files If checked, exam files are printed as they are found and validated.
Otherwise, the files are first displayed in a grid and the user has to click the Print button to start printing.

Show Previously Printed Folder Documents If checked, the Select Folder option will display exams that were previously printed with the Select Folder option.
By default, exams are moved to a special folder after printing and are not displayed.
Show Previously Printed Database Documents If selected, the Print From Database option will display previously printed exams from the database.
By default, printed exams saved in the database are not displayed.


Normally, you will only have the “Search Subfolders” button checked.

Workbench will load the exams to be printed.

During this process the Workbench will validate each file and display its current status when completed.

Wait until the “Print Status” column shows “Ready” beside the file(s) you want to print.

Check the box(es) beside the file(s) you wish to print and click the "Print" button (bottom right).
